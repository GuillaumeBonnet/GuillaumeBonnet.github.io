freezer-practice

A website for mobile only that displays freezer and aliments

The real site has a cold start because it is hosted on a free plan on heroku. It sleeps when there is no web traffic in a 30-minute period.

Demonstration giff
Real site in production
Features
  • Secure authentication where passwords are saved hashed.
  • Email verification and password reset through email messages
  • Collection of freezers which each contain their own collections of aliments.
  • Create, remove or edit any freezers or aliments.
  • Integration of custom svg icons.
