Salesforce Data Copy

Desktop app that clones Salesforce records and their related data from one sandbox to another.

The process is piloted by a Graphical User Interface displaying the data graph in real time.

Details
  • Authentication is handled by Salesforce CLI and set up through vscode or command lines.
  • In the First step you have to chose a query to pick your initials records in the source org.
  • In step "extract records" the app crawls the salesforce database of the source org to build the data graph of all the records that are ancestors of the initials records.
  • In step "upsert records" the data graph is upserted in the target org's salesforce database in the correct order to satisfy dependencies between records.
