Meta Movie

A browser plugin for Chrome that gives you facts on the form of text bubbles in Netflix videos

Features

On a movie/episode page you can select one of the available decks of fact cards.

To create a deck you need to log in. The passwords are hashed in database and the communications are done over Secure Https.
